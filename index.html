Recommended Salesforce Diagramming Solutions for Windows

Diagrams

When it comes to modeling trustable data with Salesforce, there are several types of diagrams that can be useful:

  • Entity Relationship Diagrams (ERD)
  • Process Flow Diagrams
  • UML Diagrams
  • System Architecture Diagrams
  • Wireframes

For more information and examples of these diagrams, you can visit the Salesforce Architect website.

Templates

It's important to start with a good foundation when creating diagrams. Salesforce provides a list of diagram templates that you can use as a starting point:

  • Entity Relationship Diagram Template
  • Process Flow Diagram Template
  • UML Diagram Template
  • System Architecture Diagram Template
  • Wireframe Template

Tools

There are several diagramming tools available for Windows that can be used to create Salesforce diagrams:

  • Lucidchart
  • Gliffy
  • Draw.io
  • SmartDraw
  • Creately
